# 🚀 TaskFlow

A modern, minimal, and high-performance task management application designed for clarity, speed, and focus.

> Stop managing tasks. Start executing them.

---
LIVE LINK:- https://taskflow-pi-ruddy.vercel.app/

## ✨ Vision

TaskFlow isn’t just another to-do app.

It’s built around a simple idea:  
**Tasks should flow — not pile up.**

Instead of cluttered lists, TaskFlow helps you:
- Focus on what matters now
- Reduce decision fatigue
- Move tasks seamlessly across states

---

## 🧠 Core Philosophy

- **Minimalism over noise**
- **Speed over features**
- **Flow over friction**

---

## ⚡ Features

- ✅ Create, edit, and delete tasks
- 📌 Smart task organization
- 🔄 Smooth task transitions (flow-based UX)
- 🎯 Focus-first interface (no distractions)
- ⚡ Instant UI interactions
- 🧩 Modular and scalable architecture

## 🚀 Getting Started

### 1. Clone the repo
```bash
git clone https://github.com/Suku8008m/TaskFlow.git
cd TaskFlow
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the app
```bash
npm run dev
```
